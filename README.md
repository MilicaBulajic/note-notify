## Technologies

| Technology          | Use                                                 |
| ---------------     | --------------------------------------------------- |
| React+TypeScript(Vite)     | Javascript Library for UI                           |
| React Hooks         | Reusuable stateful logic                            |
| Bootrstrap   |  UI Framework                                  |
| React Router     | Library for routing in React                                          |
